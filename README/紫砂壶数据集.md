# 紫砂壶数据集

## 数据集简介

该数据集包含了多种类型紫砂壶的图片和相关信息，旨在为紫砂壶的学术研究提供支持。数据集涵盖了不同风格和类型的紫砂壶，包括几何形、自然形、花器和提梁壶等。

## 紫砂壶类型介绍

### 1. 几何形紫砂壶

几何形紫砂壶以简洁的几何形状为基础，如圆形、方形、球形、筒形等。这类壶型注重比例和对称，线条流畅，体现了现代设计的简约美学。常见的几何形紫砂壶有仿鼓壶、石瓢壶、仿古壶等。

### 2. 自然形紫砂壶

自然形紫砂壶模仿自然界中的各种形态，如动植物、山水等。这类壶型造型生动，富有创意，往往将自然元素与实用功能完美结合。常见的自然形紫砂壶有南瓜壶、竹节壶、莲子壶等。

### 3. 花器

花器是紫砂壶中的一种特殊类型，以花卉为主题进行设计和制作。花器的造型多样，色彩丰富，常常采用浮雕、透雕等工艺，使花卉图案栩栩如生。常见的花器有牡丹壶、梅花壶、荷花壶等。

### 4. 提梁壶

提梁壶是一种带有提梁的紫砂壶，提梁通常位于壶身的上方，方便使用者提拿。提梁壶的造型多样，提梁的设计也各不相同，有的简约大方，有的则精美华丽。常见的提梁壶有东坡提梁壶、虚扁提梁壶、南瓜提梁壶等。

## 数据来源

该数据集的数据来源于以下两个渠道：

1. **江苏省宜兴紫砂工艺厂**：提供了部分紫砂壶的高清图片和详细信息，包括壶型、尺寸、作者、制作时间等。
2. **网络收集**：从互联网上收集了大量的紫砂壶图片和相关信息，经过筛选和整理后纳入数据集。

## 数据集下载

### 前提条件

确保已安装 git-xet（参考官方文档：[hf.co/docs/hub/git-xet](https://hf.co/docs/hub/git-xet)）

### 安装 git-xet 命令

```bash
brew tap huggingface/tap
brew install git-xet
git xet install
```

### 克隆数据集（完整下载所有文件）

```bash
git clone https://huggingface.co/datasets/AGI-FBHC/ChaHu
```

### 备选方案：仅克隆数据集（不下载大文件，仅保留指针）

```bash
GIT_LFS_SKIP_SMUDGE=1 git clone https://huggingface.co/datasets/AGI-FBHC/ChaHu
```
